Local Docker Setup for Web App

Containerized a simple Node.js application with Docker, created a Dockerfile, and ran it locally.

Problem Statement:

"It works on my machine" issue when sharing code with different environments.

My Solution:

  • Created a Dockerfile with Node.js base image
  • Defined environment variables in container
  • Mapped container ports to host machine
  • Used Docker Compose for multi-container setup

Technical Implementation:

# Dockerfile
FROM node:14-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
EXPOSE 3000
CMD ["npm", "start"]

Tools Used:

Docker Node.js Docker Compose
